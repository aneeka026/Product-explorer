# SPA Project - Product Explorer

A **Single Page Application (SPA)** built with **React.js** that fetches and displays product data from [FakeStoreAPI](https://fakestoreapi.com).  
The app includes routing, reusable components, loading/error states, and a responsive UI.



##  Features

- **API Integration**: Fetches products dynamically from FakeStoreAPI.
- **Routing**: 
  - `/` → Home Page  
  - `/products` → Product List Page  
  - `/products/:id` → Product Detail Page
  -  `/about` → About Page
  -  `/contact` → Contact Page
  -  `/for-men` → Men's Clothing Page
  -  `/for-women` → Women's Clothing Page
  -  `/accessories` → Accessories Page
  -  `/wishlist` → Wishlist Page
- **Reusable Components**: Navbar, Footer, Card, CartDrawer, Loader, Error messages.
- **State Management**: React hooks (`useState`, `useEffect`) and localStorage for cart & wishlist.
- **Responsive Design**: Mobile, tablet, and desktop friendly.
- **Wishlist & Cart**: Add/remove items with localStorage persistence.
- **Search**: Search bar to filter products dynamically.
- **Loading & Error States**: Handles API fetching gracefully.



##  Tech Stack

- **Frontend**: React.js (Vite)
- **Routing**: `react-router-dom`
- **State Management**: React Hooks + LocalStorage
- **Icons**: `react-icons`
- **Styling**: CSS Flexbox/Grid, responsive media queries
- **API**: [FakeStoreAPI](https://fakestoreapi.com)
